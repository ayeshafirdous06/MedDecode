# **App Name**: MedDecode

## Core Features:

- Report Upload and Processing: Allows users to upload medical reports (PDF or image) for AI analysis.
- Language Selection: Enables users to choose their preferred language (Hindi, English, Telugu) for report explanations.
- AI-Powered Report Explanation: Generates explanations of medical reports in simple, non-technical language, tailored to the selected language and using color indicators for informational, needs attention, and discuss with doctor categories. LLM may use reasoning tool.
- Doctor Summary Generation: Creates a concise summary of the report, highlighting key findings and suggesting potential discussion points with a healthcare professional.  LLM may use reasoning tool.
- Lifestyle Awareness Information: Provides general lifestyle awareness information relevant to the report findings, without offering medical advice or suggesting specific treatments.
- Disclaimer Display: Shows a mandatory disclaimer, clarifying the app's educational purpose and advising users to consult healthcare professionals.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) for a trust-based medical theme.
- Background color: Light gray (#F5F5F5) to create a clean, minimal interface.
- Accent color: Light green (#90EE90) to highlight informational content.
- Font: 'PT Sans', a humanist sans-serif font that combines a modern look and a little warmth or personality; suitable for both headlines and body text.
- Mobile-first design with clear headings, bullet points, and short paragraphs for readability on smaller screens.
- Use medical-themed icons and illustrations in a clean and friendly style.
- Subtle transitions and animations to guide users through the report explanation process.
